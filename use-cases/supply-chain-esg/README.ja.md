# ユースケース: サプライチェーン ESG

## テーゼ

**申告される ≠ 証明される**

CBAM・EUDR・DPP（デジタル製品パスポート）は炭素排出・労働条件・原産地の証明を求めますが、現状は自己申告と紙のサプライヤ書類に依存します。多階層を遡る検証は事実上不可能で、規制対応はリスクと工数の塊になります。Lemmaは各サプライヤの発行属性を署名付きで連鎖させ、データを開示せずに規制適合を証明する経路を作ります。
