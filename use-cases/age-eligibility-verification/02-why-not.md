---
title: "Why existing tools fall short"
---


Three at once: prove age/eligibility without revealing the birth date; let stores/auditors verify independently; make it tamper-proof.

| Tool | Prove without revealing | Independently verifiable | Tamper-proof |
| :-- | :-: | :-: | :-: |
| Showing/copying an ID | ✗ | △ | ✗ |
| Self-declared button | ✗ | ✗ | ✗ |
| Signed PDF | ✗ | △ | ✓ |
| **ZK proof (Lemma)** | **✓** | **✓** | **✓** |

ID copies store the birth date; self-declaration can't be verified. Only a ZK proof does all three. **Only work requiring all three at once is Lemma's territory.**
