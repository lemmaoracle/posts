# 関連ユースケース

## X402 Commerce

**Pillar:** P3 Agent Trust Chain

ワークフローのサブエージェントがx402レール上で購入を開始するとき、各エージェントの権限が証明されなければならない。マルチエージェントワークフローの証明が、x402の売り手が検証する委任の連鎖を提供する。

→ [X402 Commerce](/ja/use-cases/x402-commerce/)

## Delegated Treasury

**Pillar:** P3 Agent Trust Chain

オーケストレータが購買権限をサブエージェントに委任するとき、支出制御アテステーションは委任と共に伝わらなければならない。Delegated Treasuryのアテステーションとワークフローの証明が組み合わさって、権限とスコープの両方を証明する。

→ [Delegated Treasury](/ja/use-cases/delegated-treasury/)
