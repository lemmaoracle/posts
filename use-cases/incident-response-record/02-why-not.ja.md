---
title: "なぜ既存手段では足りないか"
---


この業務は3つが同時に要ります。顧客情報・詳細を出さず対応を証明できること。第三者（裁判・行政・監査）が独立検証できること。後から改ざんできないこと。

| 手段 | 中身を出さず証明 | 第三者が独立検証 | 改ざん不能 |
| :-- | :-: | :-: | :-: |
| 対応記録 / 台帳 | ✗ | ✗ | ✗ |
| 電子署名付き PDF | ✗ | △ | ✓ |
| 社内システムのログ | △ | ✗ | ✗ |
| **ZK 証明＋来歴（Lemma）** | **✓** | **✓** | **✓** |

台帳やログは管理者が書き換えられ、顧客情報ごと開示しがちです。3つを同時に満たすのは ZK 証明＋来歴だけ。**3つが同時に要る業務だけが Lemma の領土です。**
