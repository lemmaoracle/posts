---
title: "なぜ既存手段では足りないか"
---


この業務は3つが同時に要ります。理由を現場に出さず対応区分を渡せること。権限者・監査が独立に検証できること。誰がいつ付けたかを改ざんできないこと。

| 手段 | 理由を出さず対応を渡す | 権限者・監査が独立検証 | 改ざん不能 |
| :-- | :-: | :-: | :-: |
| 顧客台帳 / メモ欄 | △ | ✗ | ✗ |
| アクセス制御のみ | ✓ | ✗ | ✗ |
| フラグだけ手入力 | ✓ | ✗ | ✗ |
| **ZK 証明（Lemma）** | **✓** | **✓** | **✓** |

アクセス制御は「現場に理由を見せない」までは作れますが、「そのフラグが正当・改ざんされていない」を後から独立に証明できません。3つを同時に満たすのは ZK 証明だけ。**3つが同時に要る業務だけが Lemma の領土です。**
