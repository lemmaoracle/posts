---
title: "Worked example"
---


A hotel chain tags guests with past trouble using an internal code — the front terminal shows only the code, the reason is hidden, and only managers can look it up. Not seeing the reason is itself a safeguard, but there's no way to later show the code was set legitimately and untampered.

Had Lemma been in place, the front desk receives only a "handling category" as a proof, while a manager views the reason via selective disclosure. Who set the code, when, and on what basis stays tamper-proof, so complaints and subject-access requests can be answered while opening the basis only as needed.
