# 問題提起: 有効な暗号、誤った意味

2026年のクロスチェーンブリッジエクスプロイトは一つの構造的パターンを共有している：トランザクションは暗号論理的に有効だが、出所に関するクロスシステムの前提は実行前に検証されていない。受信システムは、覆すことができる——そして覆された——信頼の前提に基づいてコミットする。

Kelp DAO攻撃（2026年4月）はこれを具体的に示した：1-of-1 DVNが偽のクロスチェーンメッセージを承認したため、$292Mが流出した。署名は有効だった。メッセージは偽造されていた。攻撃後、侵害されたRPCノード上のマルウェアがログを削除し、フォレンジック証拠を消滅させた。

**暗号論理的に有効 ≠ 意味的に正しい。** それがLemmaが埋める構造的ギャップである。
