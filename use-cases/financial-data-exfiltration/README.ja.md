# ユースケース: 金融データ流出防止

## テーゼ

**アクセスログがある ≠ 改ざんされていない**

組織を跨いで顧客データにアクセスする業務において、誰が何にいつアクセスしたかの証跡を、改ざん不能な暗号的証明として残す。組織間の信頼境界が曖昧になる場面に、規制当局と当事者の双方が独立検証できる共通の真実レイヤを置く構造。
