## Why Now? — The Convergence

1. **Regulatory pressure is escalating** — FSA has moved from guidance to enforcement. Financial institutions need *verifiable* compliance, not just policy documents.

2. **Industry-wide scope** — 3,500+ cases across 4 major insurers means this is not a one-company problem. Every financial institution with secondment arrangements is exposed.

3. **Existing tools are insufficient** — DLP detects anomalies but cannot *prove* what happened. SIEM logs are mutable and siloed. Regulators need evidence that stands up to scrutiny.

4. **Cross-organization proof is a new category** — No existing tool provides cryptographic proof of data access across organizational boundaries. This is a gap Lemma fills uniquely.
