# Use Case: Financial Data Exfiltration Defense

## Thesis

**Access logs exist ≠ untampered**

In operations where customer data is accessed across organizational boundaries, Lemma preserves who accessed what and when as tamper-proof cryptographic proofs. Where trust boundaries between organizations become ambiguous, Lemma provides a shared layer of truth that regulators and parties can independently verify.
