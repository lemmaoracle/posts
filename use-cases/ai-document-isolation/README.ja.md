# ユースケース: 社内文書を AI に使わせるが生データは触らせない

## テーゼ

**AI に使わせる ≠ 生データを渡す**

機密文書や個人情報をモデルや外部に漏らさず、AI には「必要な事実・属性」だけを証明付きで渡せる構造を提供します。生の PII や原本にモデルを触れさせず、「AI が実際に何を使ったか」を後から独立検証できる証跡として残せます。
