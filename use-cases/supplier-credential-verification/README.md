# Use Case: Verify Supplier Licenses, ISO & Certificates Without Handing Over Originals

## Thesis

**Submitted ≠ valid**

Verify that a supplier "validly holds" a certification, license, or insurance — as a proof, never the original. Forgery, expiry, and reuse come off the path, and revocation (expiry, withdrawal) stays trackable so "valid when submitted but now lapsed" is detectable.
