# ユースケース: 仕入先の許認可・ISO・証書を原本を渡さず確認

## テーゼ

**提出された ≠ 有効である**

サプライヤーが「有効な認証・許認可・保険を保有している」ことだけを、原本を渡さず証明として検証できる構造を提供します。偽造・期限切れ・使い回しを外し、失効(期限切れ・取り消し)も追えるため、「提出時は有効だったが今は失効」も検知できます。
