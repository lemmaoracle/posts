---
title: "なぜ既存手段では足りないか"
---


この業務は3つが同時に要ります。中身(原本)を渡さず「有効」を証明できること。組立側・監査が独立に検証できること。偽造・改ざんできないこと。

| 手段 | 原本を渡さず証明 | 組立側・監査が独立検証 | 偽造・改ざん不能 |
| :-- | :-: | :-: | :-: |
| 証書 PDF のやり取り | ✗ | ✗ | ✗ |
| 電子署名付き PDF | ✗ | △ | ✓ |
| 認証機関への都度照会 | △ | △ | ✓ |
| **ZK 証明(Lemma)** | **✓** | **✓** | **✓** |

PDF は偽造・改ざんでき、署名付きでも中身が露出します。3つを同時に満たすのは ZK 証明だけ。1つでも欠けてよいなら既存手段で十分です。**3つが同時に要る業務だけが Lemma の領土です。**
