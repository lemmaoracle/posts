# Use Case: EUDR Traceability

## Thesis

**Traced ≠ provable**

EUDR demands due diligence on country of origin, harvest date, and permit status for every shipment into the EU. The document-based workflow conflates "traced" with "provable" — a trader can reproduce paperwork without proving its referent. Lemma issues ZK attestations against each supplier's source data so the importer's due-diligence statement is a cryptographic chain, not a stack of declarations.
