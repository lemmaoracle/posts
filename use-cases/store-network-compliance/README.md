# Use Case: Store-Network License, Hygiene & Insurance Compliance

## Thesis

**Show conformance, not the originals.**

Collecting certificate PDFs from many stores or franchisees carries forgery, expiry, and reuse — "submitted" and "valid now" drift apart. Lemma lets each store present only an issuer-signed proof that the required licenses, food-hygiene, and insurance are validly held; originals never travel, and revocation stays trackable.
