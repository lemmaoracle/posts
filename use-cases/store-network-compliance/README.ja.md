# ユースケース: 店舗網の許認可・衛生・保険の適合確認

## テーゼ

**適合は示す。原本は出さない。**

多数の店舗・加盟店から証書 PDF を集める運用は、偽造・期限切れ・使い回しを抱え、「提出された」と「いま有効である」が乖離します。Lemma は各店が「必要な許認可・食品衛生・保険を有効に保有」だけを発行体署名付きで証明として提示し、原本は渡しません。失効も追える構造です。
