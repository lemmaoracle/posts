---
title: "なぜ既存手段では足りないか"
---


この業務は3つが同時に要ります。原本を渡さず「有効」を証明できること。本部・監査が独立に検証できること。偽造・改ざんできないこと。

| 手段 | 原本を渡さず証明 | 本部・監査が独立検証 | 偽造・改ざん不能 |
| :-- | :-: | :-: | :-: |
| 証書 PDF の収集 | ✗ | ✗ | ✗ |
| 電子署名付き PDF | ✗ | △ | ✓ |
| 発行元への都度照会 | △ | △ | ✓ |
| **ZK 証明（Lemma）** | **✓** | **✓** | **✓** |

PDF は偽造・期限切れが紛れ、照会は手間です。3つを同時に満たすのは ZK 証明だけ。**3つが同時に要る業務だけが Lemma の領土です。**
