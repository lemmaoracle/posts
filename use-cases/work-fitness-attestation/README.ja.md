# ユースケース: 就業適格（健康・研修・資格）の証明

## テーゼ

**適格は示す。健康情報は出さない。**

就業に健康確認・研修・資格が要る現場では、検便結果や健診情報まで集めれば、その保管が要配慮個人情報の漏洩面になります。Lemma は「必要な健康確認・研修・資格を満たす（就業適格）」だけを発行体署名付きで証明として提示する構造を提供します。健診結果・受講履歴は本人と発行体に留まり、配属・監査は中身を見ずに検証できます。
