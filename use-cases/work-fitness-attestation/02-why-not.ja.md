---
title: "なぜ既存手段では足りないか"
---


この業務は3つが同時に要ります。健康情報を出さず適格を証明できること。配属・監査が独立に検証できること。後から改ざんできないこと。

| 手段 | 健康情報を出さず証明 | 第三者が独立検証 | 改ざん不能 |
| :-- | :-: | :-: | :-: |
| 健診結果 / 研修記録の収集 | ✗ | △ | ✗ |
| 名簿 / Excel | ✗ | ✗ | ✗ |
| 電子署名付き PDF | ✗ | △ | ✓ |
| **ZK 証明（Lemma）** | **✓** | **✓** | **✓** |

健診・研修記録の収集は要配慮個人情報ごと保管され、その保管が漏洩面になります。3つを同時に満たすのは ZK 証明だけ。**3つが同時に要る業務だけが Lemma の領土です。**
