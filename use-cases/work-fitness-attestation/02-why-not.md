title: "Why existing tools fall short"
---


Three at once: prove fitness without exposing health data; let assignment/auditors verify independently; make it tamper-proof.

| Tool | Prove without health data | Independently verifiable | Tamper-proof |
| :-- | :-: | :-: | :-: |
| Collecting medical/training records | ✗ | △ | ✗ |
| Roster / spreadsheet | ✗ | ✗ | ✗ |
| Signed PDF | ✗ | △ | ✓ |
| **ZK proof (Lemma)** | **✓** | **✓** | **✓** |

Collecting records stores special-category data, and that storage is a leakage surface. Only a ZK proof does all three. **Only work requiring all three at once is Lemma's territory.**
