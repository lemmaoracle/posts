# Use Case: Public Procurement Attestation

## Thesis

**Submitted ≠ proven eligible**

A public-procurement evaluator's job is to verify, not to read. Today, document submission conflates the two — every certification, balance sheet, and reference letter passes in raw form through the evaluator's office, expanding the disclosure surface for both the bidder and the buyer. Lemma issues per-attribute ZK attestations so the evaluator checks "the bidder meets the qualification predicate" against a cryptographic proof, while the underlying documents stay with the bidder.
