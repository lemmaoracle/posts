# 関連ユースケース

## Delegated Treasury

**Pillar:** P3 Agent Trust Chain

買い手エージェントの支出権限が企業から委譲されている場合、売り手は売り手の属性だけでなく、買い手の支払い権限も検証できる。オンチェーンの支出制御アテステーションが信頼の連鎖を完成させる。

→ [Delegated Treasury](/ja/use-cases/delegated-treasury/)

## Multi-Agent Workflows

**Pillar:** P3 Agent Trust Chain

オーケストレータエージェントが購買エージェントを派遣するとき、委任の連鎖は追跡可能でなければならない。Lemmaは誰が購入を許可し、各ステップでどのスコープが委任されたかを証明する。

→ [Multi-Agent Workflows](/ja/use-cases/multi-agent-workflows/)
