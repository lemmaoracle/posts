# ユースケース: 有資格者の配置・安全教育の証明

## テーゼ

**配置された ≠ 有資格である**

その作業を有資格・安全教育済みの人が行ったことを、本人の資格履歴を出さずに証明。元請・監査は「有資格である」だけを検証でき、後から改ざんもできない。
