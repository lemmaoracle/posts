# ユースケース: 長期契約の記録・見積の証明

## テーゼ

**記録がある ≠ 証明できる**

15〜20 年に及ぶ保守契約・インフラ台帳で、後年に「当時いくらだったか」「どの仕様で合意したか」「点検は本当に行われたか」を問われたとき、原本を開示せず「その時、正当に存在し・正当に修正された」ことを暗号的に証明できる構造を提供します。
